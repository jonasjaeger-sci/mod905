Release Notes

This is version |version| of pyretis.


Upgrading
=========

To upgrade pyretis to the latest version, use ``pip``:

::

    pip install -U pyretis

You can determine your currently installed version using ``pip freeze``:

::

    pip freeze | grep pyretis
