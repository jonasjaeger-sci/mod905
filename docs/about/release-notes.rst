Release Notes

This is version |version| of pytismol.


Upgrading
=========

To upgrade pytismol to the latest version, use ``pip``:

::

    pip install -U pytismol

You can determine your currently installed version using ``pip freeze``:

::

    pip freeze | grep pytismol
