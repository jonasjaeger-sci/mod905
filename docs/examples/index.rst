########
Examples
########

We have prepared several examples to show the use of pyretis.
All examples are included in the pyretis/examples folder.
Some example simulations include:

-  :ref:`Umbrella sampling <examples-mc-umbrella-sampling>` in a one dimensional potential
-  :ref:`Molecular dynamics <examples-md>` of a simple Lennard-Jones fluid
-  Calculation of a rate constant using transition interface sampling
-  Calculation of a rate constant using replica exchange transition interface sampling
-  Making use of the :ref:`pyretis API <api-doc>` to define
   :ref:`custom order parameters <examples-modify-orderparameter>`
